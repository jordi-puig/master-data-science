# PAC 1: Métodos de Aprendizaje por Refuerzo Tabular

Este directorio contiene el enunciado y la resolución de la **PAC1** (Prueba de Evaluación Continua 1). 

## Contenido

El objetivo principal de esta evaluación es dominar los primeros algoritmos de Aprendizaje por Refuerzo Clásico sin utilizar Redes Neuronales (Métodos Tabulares).

1. **Jupyter Notebooks**: El fichero con el enunciado y solución principal `_PAC1-Sol.ipynb` con todo el desarrollo matemático algorítmico y explicaciones de cada ejercicio que aplican métodos de **Montecarlo (On-policy / Off-policy)**.
2. **Entornos Custom (Gym)**: Implementación de entornos personalizados como `blackjack_env.py`.
3. **Scripts Auxiliares**:
   * `ex1.py`, `ex2.py`: scripts en Python con los modelos para ejecutar en local o testear el entorno.
   * `ex3-*.py`: Ejemplos de exploración vs explotación.
   * `ex4.py`, `ex4b.py`.
