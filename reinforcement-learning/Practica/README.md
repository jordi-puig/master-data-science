# Práctica Final: Reinforcement Learning (Data Science UOC)

Este directorio alberga todo el código fuente correspondiente a la **Práctica Final** o Trabajo final dedicado exclusivamente al Módulo de *Aprendizaje por Refuerzo*.

## Resumen del Proyecto
El proyecto engloba todo conocimiento adquirido a lo largo de las PACs (Q-Learning clásico, DQN, Arquitecturas Multicapa o CNN, y Actor-Crítico) para resolver y estabilizar un entorno completo con la librería de simulaciones de *OpenAI Gym/Gymnasium*.

Entre las metas principales del análisis destacan el fine-tuning de hiperparámetros de las Redes (*learning_rate, epsilon, gamma, batches...*), validaciones exhaustivas (plot del performance o de los rewards móviles superando la cota base), y la presentación o serialización de los modelos exportados finales (`.pth` de PyTorch o modelos TF).
